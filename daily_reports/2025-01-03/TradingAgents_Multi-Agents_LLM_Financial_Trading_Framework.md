# TradingAgents：多智能体 LLM 金融交易框架
发布时间：2024年12月28日

`Agent应用`
> TradingAgents: Multi-Agents LLM Financial Trading Framework
>
> 在利用由大型语言模型（LLMs）驱动的智能体社会实现自动问题解决方面，已取得显著成果。在金融领域，工作重点多为处理特定任务的单智能体系统，或独立收集数据的多智能体框架。然而，多智能体系统复制现实世界交易公司协作动态的潜力尚未充分挖掘。TradingAgents 提出了一个受交易公司启发的全新股票交易框架，其中包含由 LLM 驱动、担任专门角色（如基本面分析师、情绪分析师、技术分析师和具有不同风险特征的交易员）的智能体。该框架涵盖了评估市场状况的多头和空头研究智能体、监控风险敞口的风险管理团队，以及综合辩论和历史数据中的见解以做出明智决策的交易员。通过模拟动态、协作的交易环境，此框架旨在提升交易表现。详细的架构和大量实验表明其优于基线模型，在累计回报、夏普比率和最大回撤方面有明显改进，凸显了多智能体 LLM 框架在金融交易中的潜力。
>
