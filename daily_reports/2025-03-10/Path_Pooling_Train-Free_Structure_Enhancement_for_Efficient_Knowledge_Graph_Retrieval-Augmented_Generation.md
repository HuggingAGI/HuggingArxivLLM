# # 路径池化：无训练结构增强助力高效知识图谱检索增强生成
发布时间：2025年03月07日


> Path Pooling: Train-Free Structure Enhancement for Efficient Knowledge Graph Retrieval-Augmented Generation
>
> 大型语言模型虽在诸多任务中表现出色，但现实应用中仍面临生成不实信息与知识匮乏的挑战。为解决这一问题，众多基于知识图谱的检索增强生成方法应运而生，它们通过整合知识图谱中的结构与语义信息来增强模型输出的准确性和可靠性。然而，现有方法在结构信息的高效利用上仍有不足，往往面临计算开销过大或知识利用率低的困境。受图表示学习中平滑操作的启发，我们提出了一种简单且无需训练的创新策略——路径池化。该方法通过独特的基于路径的池化操作引入结构信息，并能与现有KG-RAG方法无缝对接，实现即插即用，从而更充分地挖掘结构信息的价值。大量实验结果表明，将路径池化整合到前沿的KG-RAG方法中，能够在各种应用场景下显著提升模型性能，同时几乎不增加额外计算成本。我们的代码即将在GitHub上开源，欢迎关注：https://github.com/hrwang00/path-pooling.
>
