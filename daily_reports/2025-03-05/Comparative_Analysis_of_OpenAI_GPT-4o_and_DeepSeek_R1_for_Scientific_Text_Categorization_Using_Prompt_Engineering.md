# 基于提示工程的 OpenAI GPT-4o 与 DeepSeek R1 科学文本分类能力对比研究
发布时间：2025年03月03日


> Comparative Analysis of OpenAI GPT-4o and DeepSeek R1 for Scientific Text Categorization Using Prompt Engineering
>
> 本研究探讨了大型语言模型如何通过提示工程对科学论文中的句子进行分类。我们采用了两个先进的基于网络的模型，GPT-4o（由OpenAI开发）和DeepSeek R1，将句子分类到预定义的关系类别中。尽管DeepSeek R1已经在其技术报告中通过了基准数据集的测试，但其在科学文本分类方面的表现尚未被探索。为了解决这一空白，我们引入了一种新的专门为此任务设计的评估方法。我们还整理了一个来自不同领域的经过清洗的科学论文数据集，为比较这两个模型提供了一个平台。利用这个数据集，我们分析了它们在分类方面的有效性和一致性。
>
