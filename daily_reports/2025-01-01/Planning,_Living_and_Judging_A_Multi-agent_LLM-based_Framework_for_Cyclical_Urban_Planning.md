# 规划、生活与评判：基于多智能体 LLM 的周期性城市规划框架
发布时间：2024年12月29日

`Agent应用`
> Planning, Living and Judging: A Multi-agent LLM-based Framework for Cyclical Urban Planning
>
> 在城市化进程中，城市更新面临着重大挑战，需要采取适应性策略来满足不断变化的需求。借助大型语言模型（LLMs）的发展，我们提出了周期性城市规划（CUP）这一新模式，它能在闭环中持续生成、评估和优化城市规划。具体而言，我们基于多智能体LLM的框架包含三个关键部分：（1）规划，LLM智能体依据上下文数据生成和改进城市规划；（2）生活，智能体模拟居民的行为与互动，为城市环境中的生活建模；（3）判断，涉及评估规划效果并提供迭代反馈以促进改进。这种循环过程造就了一种动态且具有响应性的规划方式。在真实世界数据集上的实验表明，我们的框架作为连续且适应性的规划过程是有效的。
>
