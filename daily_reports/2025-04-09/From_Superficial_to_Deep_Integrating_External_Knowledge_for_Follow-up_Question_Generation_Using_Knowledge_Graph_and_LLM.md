# # 标题 从浅到深：结合知识图谱与LLM，融合外部知识生成后续问题
发布时间：2025年04月08日


> From Superficial to Deep: Integrating External Knowledge for Follow-up Question Generation Using Knowledge Graph and LLM
>
> 在对话系统中，动态生成后续问题能够帮助用户探索信息，提供更佳的用户体验。人类通常能够提出涉及常识性知识并展现高级认知能力的问题，但现有方法生成的问题往往局限于浅层次的上下文问题，缺乏启发性，与人类水平存在较大差距。为此，本文提出了一种三阶段的外部知识增强型后续问题生成方法：首先识别上下文主题，然后在线构建知识图谱（KG），最后结合大型语言模型生成最终问题。通过引入外部常识知识并进行知识融合操作，该模型能够生成信息丰富且具有探索性的后续问题。实验结果表明，与基线模型相比，我们的方法生成的问题更具信息量，更接近人类提问水平，同时保持了良好的上下文相关性。
>
