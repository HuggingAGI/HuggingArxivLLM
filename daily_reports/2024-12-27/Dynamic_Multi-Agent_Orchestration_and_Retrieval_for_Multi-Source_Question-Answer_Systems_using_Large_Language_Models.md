# 利用大型语言模型的多源问答系统的动态多智能体编排与检索
发布时间：2024年12月23日

`RAG`
> Dynamic Multi-Agent Orchestration and Retrieval for Multi-Source Question-Answer Systems using Large Language Models
>
> 我们提出了一种融合大型语言模型（LLM）检索中多项先进技术的方法，以助力开发强大的多源问答系统。此方法通过协同的多智能体编排与动态检索手段，整合来自包括非结构化文档（PDF）和结构化数据库等不同数据源的信息。我们的方法借助诸如 SQL 智能体、检索增强生成（RAG）智能体和路由器智能体等专门智能体，依据每个查询的特性动态选取最适宜的检索策略。为进一步提升准确性和上下文关联度，我们运用了动态提示工程，能够实时适配特定查询的上下文。该方法在合同管理领域展现出有效性，在此领域，复杂的查询往往需要非结构化与结构化数据的无缝交互。我们的成果表明，这种方式提高了响应的准确性和相关性，为开发能在不同领域和数据源中运行的问答系统提供了一个灵活且可扩展的框架。
>
