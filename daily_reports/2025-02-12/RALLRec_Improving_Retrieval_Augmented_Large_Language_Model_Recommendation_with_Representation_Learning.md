# RALLRec: 利用表示学习提升检索增强型大规模语言模型的推荐效果
发布时间：2025年02月11日

`RAG`
> RALLRec: Improving Retrieval Augmented Large Language Model Recommendation with Representation Learning
>
> 大型语言模型（LLMs）已被整合到推荐系统中，以增强对用户行为的理解。检索增强生成（RAG）技术进一步被引入这些系统，以检索更相关项目并提升系统性能。然而，现有RAG方法主要依赖文本语义，往往未能整合最相关项目，限制了系统效果。

在本文中，我们提出了一种针对检索增强大型语言模型推荐的表征学习方法（RALLRec）。具体而言，我们通过提示LLMs生成更详细的产品描述来增强文本语义，随后进行文本和协作语义的联合表征学习，分别由LLM和推荐模型提取。考虑到用户兴趣可能存在的时序变化特性，我们进一步引入了一种简单而有效的重排序方法，以捕捉用户偏好的动态变化。我们在三个真实数据集上进行了广泛实验，评估结果验证了我们方法的有效性。代码已公开发布于https://github.com/JianXu95/RALLRec。
>
> https://arxiv.org/abs/2502.06101
