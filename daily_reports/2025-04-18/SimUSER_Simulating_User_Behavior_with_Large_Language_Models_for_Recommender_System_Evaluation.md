# # 摘要
最近大型语言模型（LLMs）的突破性进展引领了一场从机器人流程自动化到智能体流程自动化的革命性转变，这一转变通过基于LLMs自动化工作流编排过程实现了。
发布时间：2025年04月17日

`推荐系统`
> SimUSER: Simulating User Behavior with Large Language Models for Recommender System Evaluation
>
> 推荐系统在众多实际应用场景中发挥着核心作用，但其性能评估仍是一项重大挑战，原因在于离线指标与线上行为之间存在差距。鉴于真实用户数据的稀缺性及其限制（例如隐私问题），我们引入了SimUSER，这是一个充当可信且成本低廉的人类代理的框架。SimUSER首先从历史数据中识别出自洽的人设，通过为用户档案添加独特的背景和个性特征来丰富用户画像。然后，本评估的核心在于配备有角色、记忆、感知和思维模块的用户，与推荐系统进行交互。与以往工作相比，SimUSER在微观和宏观层面上都更贴近真实人类行为。此外，我们还进行了富有洞察力的实验，探讨缩略图对点击率的影响、曝光效应以及评论对用户互动的影响。最后，我们根据离线A/B测试结果优化推荐系统参数，从而在现实世界中提升了用户互动水平。
>
> https://arxiv.org/abs/2504.12722
