# 知识驱动的多智能体框架，助力自动化的软件架构设计
发布时间：2025年03月26日

`Agent应用`
> Knowledge-Based Multi-Agent Framework for Automated Software Architecture Design
>
> 架构设计是软件开发中至关重要的一环。然而，构建高质量架构通常需要投入大量的人力和时间成本。近年来，基于大型语言模型（LLMs）的智能体在软件工程领域取得了显著成果，但将其应用于自动化架构设计仍处于探索阶段。为此，我们提出了一个基于知识的多智能体架构设计（MAAD）框架。该框架通过模拟传统架构设计中的人类角色，借助智能体实现设计流程的自动化。MAAD整合了来自现有系统设计、权威文献和架构专家的三方知识，以增强智能体的能力。我们的目标是通过MAAD框架推动应用程序级系统开发的全面自动化。
>
