# Rollama，一个借助Ollama实现的R语言包，用于调用和利用生成式大型语言模型。
`动手训练`
> rollama，一个R包，轻松封装Ollama API，让你在本地轻松驾驭各类生成型大型语言模型（GLLM）。它和配套的学习资料致力于简化用开源模型注释文本或图像数据的过程，以及进行文档嵌入。用户还可拓展rollama，实现OpenAI API所支持的更多功能，享受更私密、可复现且无需花费的体验。
